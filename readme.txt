Replication Project

Overview

This project includes code to perform various Natural Language Processing (NLP) tasks using existing and custom datasets. The project is organized into two main folders: `codes` and `custom`.

- `codes` Folder: Contains code that runs on the provided existing datasets.
- `custom` Folder: Contains code fine-tuned specifically for the custom datasets created for this project.

Folder Structure

project-root/
│
├── codes/
│   └── codes.py
│
├── custom/
│   └── custom.py
│
├── dataset/
│   ├── existing_datasets/
│   └── custom_datasets/
│
├── report.tex
│
└── README.txt

- `codes/`: This folder includes code scripts that operate on the datasets provided to us.
  
- `custom/`: This folder contains code that has been fine-tuned to work specifically with the custom datasets. These custom datasets are located in the `dataset/custom_datasets` folder.
